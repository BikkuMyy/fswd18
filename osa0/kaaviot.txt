title Navigointi muistiinpanosivulle

kayttaja->selain:
note left of selain
 kayttaja klikkaa linkkiä "muistiinpanot"
end note
selain->palvelin: GET https://fullstack-exampleapp.herokuapp.com/notes
note left of palvelin
 muodostetaan sivun HTML, jossa muistiinpanot listattuna
end note
palvelin->selain: status 200, sivun HTML-koodi
selain->palvelin: GET https://fullstack-exampleapp.herokuapp.com/main.css
palvelin->selain: status 200, sivun CSS-tyylitiedosto
selain->palvelin: https://fullstack-exampleapp.herokuapp.com/main.js
palvelin->selain: status 200, sivun JS-skriptitiedosto
selain->palvelin: https://fullstack-exampleapp.herokuapp.com/data.json
palvelin->selain: status 200, sivun muistiinpanodatan sisältävä JSON-tiedosto
note left of selain
 selain näyttää palvelimen palauttaman HTML:n, 
 jossa käytetään palvelimelta haettuja tyyli-, skripti-, ja datatiedostoja. 
end note

title Muistiinpanon lisääminen

kayttaja->selain:
note left of selain
 kayttaja kirjoittaa uuden muistiinpanon lomakkeen
 tekstikenttään ja klikkaa "Talleta"-nappia
end note
selain->palvelin: POST https://fullstack-exampleapp.herokuapp.com/new_note
note left of palvelin
 lomakkeelle kirjoitettu teksti lähetetään palvelimelle
end note
palvelin->selain: status 302, uudelleenohjauspyyntö
note left of palvelin
 uudelleenohjauspyyntö lataa muistiinpanot listaavan sivun uudestaan
end note
selain->palvelin: GET https://fullstack-exampleapp.herokuapp.com/notes
palvelin->selain: status 200, sivun HTML-koodi
selain->palvelin: GET https://fullstack-exampleapp.herokuapp.com/main.css
palvelin->selain: status 200, sivun CSS-tyylitiedosto
selain->palvelin: https://fullstack-exampleapp.herokuapp.com/main.js
palvelin->selain: status 200, sivun JS-skriptitiedosto
selain->palvelin: https://fullstack-exampleapp.herokuapp.com/data.json
palvelin->selain: status 200, sivun muistiinpanot sisältävä JSON-tiedosto
note left of selain
 selain näyttää uudelleenladatun sivun,
 jossa nyt myös lisätty muistiinpano
end note
